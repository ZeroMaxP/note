```js
获取窗口的内部的宽度和高度（包括滚动条但不包括控制台）

    innerWidth      
    innerHeight     

获取文档的可视区域宽高 （不包括滚动条和控制台）
    document.documentElement.clientWidth
    document.documentElement.clientHeight

ele.clientWidth/clientHeight
    padding + 样式宽/样式高
ele.offsetWidth/offsetHeight
    padding + border + 样式宽/样式高
    就是说元素的实际占位大小

ele.scrollWidth/scrollHeight
    获取ele的padding + width/height
    但是如果子元素超过父元素宽高时，会加上超出部分的宽高（无论父元素是否overflow:hidden）


ele.offsetTop/offsetLeft 
    获取ele元素到定位父级的top和/left方向的距离（可能包含margin）


ele.scrollTop/scrollLeft; 可读可写
    X轴/Y轴的滚动距离
    在赋值时，不能带px单位，否则报错。

获取页面的实际高度/宽度：

document.body.offsetHeight
document.body.offsetWidth
```
