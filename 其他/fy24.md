2. 节点

```
childNodes //返回所有子节点(包括文字节点)
children // 返回第一级元素节点 动态获取
nodeType //返回节点类型
// 1.元素节点  3.文字节点 8注释 2.属性节点
nodeName //返回节点名字大写	
tagName //返回元素节点的名称

getAttributeNode('attr') //获取指定节点的属性节点
document.createAttribute('attrName')//创建属性节点
ele.setAttributeNode() //添加一个属性节点

firstChild    返回第一个子节点 【在低版本IE只会返回元素节点，主流浏览器可能返回文本节点】
firstElementChild   只兼容主流浏览器【返回第一个元素节点】

lastChild     兼容性同上
lastElementChild 	兼容性同上

nextSibling   返回下一个兄弟节点  兼容性同上
nextElementSibling   兼容性同上

previousSibling  	兼容性同上
previousElementSibling 	兼容性同上

parentNode    返回父节点（没有兼容问题）

offsetParent  返回定位父级（没有兼容问题）
		
childElementCount  返回子元素节点个数 ，不兼容IE8及其以下  

```
